# Client

A client is a integration point for the bot. It is used to interact with the bot.

A client can be have multiple types of integrations.
- Web: A web client is a integration point for the bot. It is used to interact with the bot through a web interface.
- Mobile: A mobile client is a integration point for the bot. It is used to interact with the bot through a mobile application.
- Desktop: A desktop client is a integration point for the bot. It is used to interact with the bot through a desktop application.
- API: An API client is a integration point for the bot. It is used to interact with the bot through a REST API.
- And more...

A client can have multiple bots, and a bot can have multiple clients.

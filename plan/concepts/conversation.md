# Conversation
It is the conversation between the user and the bot. It stores the user's conversation history.

## Conversation Message
It is a message in the conversation. It stores the message parts (text, image, video, audio, etc), the user who sent the message, and the agent that replied.

## Parts
It is a part of the message. It stores the part type (text, image, video, audio, etc) and its content.
